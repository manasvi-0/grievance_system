## 📌 Overview

An intelligent grievance management system that uses **AI-based classification** to automatically route complaints to the correct department, track their progress, and provide analytics.

---

## ✨ Features

### 🧠 AI Smart Routing
- Automatically assigns complaints to departments
- Hybrid approach (ML + keyword logic)

### 📊 Dashboard
- User-specific complaint tracking
- Stats:
  - Total
  - Pending
  - In Progress
  - Resolved

### 🏢 Departments
- Department-wise complaint distribution
- Average resolution time

### 🛠 Admin Panel
- View all complaints of logged-in user
- Update complaint status
- Clean table UI

### 📈 Analytics
- Bar chart → department-wise complaints
- Pie chart → status distribution
- Line chart → trends over time
- AI insights

### 🔍 Complaint Details
- Full complaint view
- Department + confidence score
- Status tracking
